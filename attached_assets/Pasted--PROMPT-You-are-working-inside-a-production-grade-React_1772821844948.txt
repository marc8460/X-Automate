2. Implement Ranking Engine

Create a reusable function:

function computeCommentOpportunityScore(tweet: Tweet): number

Use this logic:

Definitions:

L = like_count
R = retweet_count
C = reply_count
Q = quote_count
V = impression_count (if available)
T = age in minutes

If impressions exist:

engagementRate = (L + 2R + 3C + 2Q) / V

Else:

engagementProxy = (L + 2R + 3C + 2Q)

velocity = engagementProxy / max(T, 1)

replyGapFactor = 1 - (C / (L + R + Q + 1))

Raw score:

score_raw =
0.4 * normalized(engagementRate or engagementProxy) +
0.4 * normalized(velocity) +
0.2 * replyGapFactor

Normalize per batch.

Scale final result:

score = Math.min(100, Math.round(score_raw * 100))

Attach:

score: number

to each returned post.
