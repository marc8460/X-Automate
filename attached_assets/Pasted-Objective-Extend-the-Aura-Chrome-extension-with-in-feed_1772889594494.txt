Objective
Extend the Aura Chrome extension with **in-feed tweet analysis and viral reply generation directly on X.com**.
Users should be able to analyze tweets, generate viral comments, customize the tone, and insert replies directly into the composer.

This feature should work **while the user scrolls their X feed**, without needing to open the Aura dashboard.

---

New Feature Overview

When a user scrolls X.com:

1. The extension detects tweet elements in the feed.
2. When hovering a tweet, an **“Analyze with Aura” button** appears.
3. Clicking it opens an **Aura analysis panel**.
4. The panel shows:

• Tweet text
• Author
• Likes
• Replies
• Retweets
• Views

The extension extracts these metrics **directly from the DOM**, without using the X API.

---

Aura Opportunity Score

The extension should display an **Aura Opportunity Score** directly on tweets.

Example:

🔥 Opportunity Score: 87

or insight badges such as:

Low reply competition
High engagement velocity

This score helps users quickly identify **tweets worth replying to**.

The score can be calculated using:

• likes
• replies
• retweets
• views
• tweet age

These metrics can be extracted from the tweet DOM.

---

Tweet Analysis Panel

Clicking **Analyze with Aura** opens a floating panel containing:

Tweet preview
Metrics
Opportunity score

Below that:

Generate Viral Replies button

When clicked, the extension sends the tweet text and metrics to the Aura backend AI.

The AI returns **5 viral reply suggestions**.

---

Custom Instruction Prompt

The panel should include a text input labeled:

Custom Instructions

Example input:

"flirty gen z girl"

The extension sends this instruction together with the tweet text to the AI:

tweet_text + custom_instruction

This allows users to control the tone of the generated replies.

---

Reply Generation Result

The panel displays 5 generated replies.

Each reply has:

Copy button
Insert Reply button

Insert Reply triggers the extension reply workflow.

---

Post Comment Workflow

When the user clicks **Insert Reply**:

1. The extension opens the tweet URL if needed.
2. The extension waits for the reply composer.
3. The generated text is inserted character-by-character.
4. If the user selected an image, the extension attaches it.

A message appears:

“Reply inserted by Aura — click Post when ready.”

The extension **does not click the Post button** (human-in-the-loop).

---

Screenshot Mode

The extension should also support **tweet screenshots**.

Users can upload a screenshot into the extension panel.

The screenshot is sent to the Aura backend for AI vision processing.

The system extracts:

tweet text
metrics if visible

Then generates viral replies in the same way as normal tweet analysis.
